Quick Overview

Mirza is a complete toolkit for managing an Apple Silicon Mac as a headless AI server from any Linux workstation. The Mac serves GGUF models locally via llama-cpp-python with Metal acceleration. You control it through the mirza CLI or this WebUI.

WebUI Views

DashboardServer status, hardware info (Chip, RAM, GPU), active model, quick actions (Wake, Sleep, Reboot, Stop LLM), and live inference logs.
ChatInteractive chat interface using the currently served model.
ModelsCloud catalog explorer with one-click download and serve capabilities.
ConfigVisual explorer of mirza.conf with refresh capability.

Primary CLI Commands

mirza statusCheck server, Llama.cpp API, and Grafana status.
mirza serve [--ctx N] [--kv-q TYPE]Start the llama.cpp inference server.
mirza stop-llmStop the active inference server.
mirza deploy <repo> [file]Download and install a model from HuggingFace.
mirza sleep / rebootRemote power management.

⚠️ Memory Safety

If a model exceeds available RAM (Weights + KV Cache + 6GB OS overhead > RAM), the card is grayed out and buttons are disabled. We won't let you crash your server by being overly optimistic.
